# Features

This folder contains documentation for the business features of the Pricing Intelligence Platform.

## What belongs here?

- **Business functionality** that users interact with
- Features with **role-based access rules**
- Descriptions of **user workflows and capabilities**
- **Not:** UI/UX design, technical implementation, or API contracts
